# 🌐 Diyana Portfolio

A modern full-stack personal portfolio website built with the MERN stack. This portfolio showcases my projects, skills, education, certifications, and professional experience while providing a CMS-powered Admin Dashboard for managing content dynamically through MongoDB.

## 🚀 Live Demo

Frontend: https://your-portfolio-url.vercel.app

## ✨ Features

* Responsive modern portfolio design
* Dynamic Hero, About, Education, Skills, Projects, and Certificates sections
* MongoDB-powered CMS
* Secure Admin Dashboard
* Project Management (Add/Edit/Delete)
* Skills Management
* Certificates Management
* Education & Experience Management
* CV Download Support
* Contact Form with Gmail Notifications
* MongoDB Atlas Database
* Vercel Deployment

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* CSS3
* React Router

### Backend

* Node.js
* Express.js
* JWT Authentication
* Nodemailer

### Database

* MongoDB Atlas
* Mongoose

### Deployment

* Vercel

## 🔐 Admin Dashboard

The Admin Dashboard allows secure management of portfolio content.

Features:

* Update Hero Section
* Update About Section
* Manage Education
* Manage Experience
* Manage Skills
* Manage Projects
* Manage Certificates
* Manage CV Information

All updates are stored in MongoDB Atlas and automatically reflected on the live portfolio website.

## 📧 Contact System

Visitors can send messages through the portfolio contact form.

Features:

* Email notifications via Gmail
* Reply-to support
* Sender details included
* Professional email formatting

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/diykumarasinghe/diyana-portfolio.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
